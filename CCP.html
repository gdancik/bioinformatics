Cancer Publication Portal (CCP)

Overview

This is the homepage of a forthcoming Cancer Publication Portal , which will organize the biomedical literature for faster querying and summarizing of cancer-related publications.

Source code

This project is a work in progress. For current efforts involving data collection and processing, see the links below.
  • https://github.com/mjunderwood/minePM - python and R scripts for data retreival and processing
  • https://github.com/mgdlxd/minePM - scripts for data retreival and storage using MongoDB
