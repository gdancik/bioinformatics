Cancer Publication Portal (CPP)

Overview

This is the homepage of the Cancer Publication Portal , a tool for searching the biomedical literature for faster querying and summarizing of cancer-related publications.

The tool currently leverages PubTator annotation and provides interactive summaries of diseases, chemicals, and gene co-occurences for a selected human gene. CPP is a work in progress and additional features will be added shortly. Feedback regarding the tool is appreciated.

A prototype of the tool is available at http://bioinformatics.easternct.edu/CPP/

Source code

Source code for the web interface is available from: https://github.com/gdancik/CPP/.
