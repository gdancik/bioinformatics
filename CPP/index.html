Cancer Publication Portal (CPP)

Overview

This is the homepage of the Cancer Publication Portal (CPP), a tool for searching and summarizing the biomedical literature based on a gene of interest, with filters available for cancer types, mutations, additional genes, and additional cancer-related terms. CPP is described in a publication in F1000Research and was presented at AACR 2019. CPP can answer questions such as

  • What tumor types are associated with EGFR?
  • What tumor mutations are associated with EGFR and erlotinib?
Following a search, users can create a PubTator collection for article viewing and curation. If you use CPP, please cite the tool using the citation below.

Citation

Dancik G.M., Williams K., Zhang M. and Romanenko N. Cancer Publication Portal: an online tool for summarizing and searching human cancer-genomic publications [version 1; peer review: 2 approved with reservations]. F1000Research 2019, 8:2073 (https://doi.org/10.12688/f1000research.21463.1)

Availability

Note: CPP was last updated with the PubTator release on 2/15/20.

Source code

CPP is open source and we welcome contributions and extensions from developers.
  • Source code for the web interface is available from: https://github.com/gdancik/CPP/.
  • Source code for setting up the underlying MySQL database is available from: https://github.com/gdancik/CPP_setup (documentation is limited, but you can contact me with questions)
  • Docker images are available for
    • the web interface: https://hub.docker.com/r/gdancik/cpp
    • the MySQL database: https://hub.docker.com/r/gdancik/dcast/
